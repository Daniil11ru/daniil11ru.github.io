Типовые сценарии атак

Ниже приведены примеры наиболее распространённых практик, которыми пользуются злоумышленники:

  • Brute Force: перебор паролей для учетных записей;
  • Phishing: подмена реального сайта с целью кражи паролей;
  • DDoS: массовые запросы на сервер для исчерпания его ресурсов;
  • RCE (Remote Code Execution): удалённое выполнение произвольного кода.

Принципы защиты

Для каждой из рассмотренных уязвимостей существуют методики предотвращения: использование параметризованных запросов, валидация входных данных, ограничение прав доступа, правильное хранение конфиденциальных данных (хеширование, шифрование), регулярное обновление ПО и т.д.
